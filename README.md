## About the Project

This is a modern, responsive, single-page campaign website designed to showcase Hon. Nawa Nawa's vision, achievements, and plans for transforming Matero Constituency. The site highlights:

- Leadership profile and background
- Vision for jobs, healthcare, infrastructure, and youth empowerment
- Photo gallery of community events and projects
- Latest news and updates
- Easy contact and engagement options

The design proudly incorporates the official UPND colors: **Red (#E31E24)**, **Green (#00853E)**, and **Gold/Yellow (#FFD700)**.

## Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth scrolling navigation with active section highlighting
- Animated counters showcasing impact
- Interactive image gallery with category filters
- Mobile-friendly hamburger menu
- Contact form (client-side confirmation)
- Back-to-top button
- Fast loading with CDN-hosted libraries
